-> Questão 1 
A história das linguagens de programação é dividida em 5 gerações:
- Linguagens de máquina;
- Linguagens assembly de baixo nível;
- Linguagens de alto nível (C, C++, Java);
- Linguagens de alto nível que se assemelham à linguagem humana (Python, SQL, Ruby).
Além da divisão histórica, existe a classificação por paradigmas:
- Linguagem não-estruturada: Permite desvio no fluxo do programa, sendo assim uma linguagem sem estrutura;
- Linguagem estruturada: Divide o programa em estruturas, podendo ser do tipo sequência, condição ou repetição;
- Linguagem orientada a objetos: Modela o programa através de objetos, com suas respectivas características e ações.

-> Questão 3
A linguagem de programação compilada passa pelo processo de conversão para código de máquina, dessa forma o desempenho tende a ser superior. Como exemplos, temos: COBOL, C, C++ e C#.

Já a linguagem de programação interpretada executa o código sem passar pelo processo de compilação, tornando o processo de execução mais custoso e lento. Exemplos: Python, JavaScript e Perl.
